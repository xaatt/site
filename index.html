Protocol Server Port
IMAP imap.purelymail.com 993
SMTP smtp.purelymail.com 465

Если вам действительно нужна качественная электронная почта без надзора корпораций и сканирования ваших писем - присоединяйтесь.

Почтовые ящики на данный момент создаю только тем кого знаю лично. После получения почтового ящика, рекомендую самостоятельно сменить пароль.
